Direct Verification Tasks (Like, Follow)

Tasks that can be verified instantly without requiring link submission

Link Verification Tasks (Comment, Quote)

Tasks that require link submission for verification

1 Initial State - Task buttons ready

Blue color indicates task is not started

2 User clicks "Comment" - Dropdown opens

Dropdown appears with "Open Twitter" and "Verify Comment" options

3 User clicks "Verify Comment" - Link submission slider appears

Link submission slider appears in dropdown, task color changes to yellow (completed)

4 Link submitted - Task verified immediately

Task name "Comment" is retained, color changes to green (verified)

Complete Flow Summary

Direct Verification Tasks

Initial: Blue "Like" button
Click: Dropdown with "Open Twitter" + "Verify Like"
Verified: Green "Like" button (name retained)

Link Verification Tasks

Initial: Blue "Comment" button
Click: Dropdown with "Open Twitter" + "Verify Comment"
Verify: Yellow "Comment" + Link submission slider
Submitted: Green "Comment" button (name retained)
